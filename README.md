🎯 Objective

The objective of this task is to learn the basics of the Pandas library, load a CSV dataset, and explore its contents by viewing rows, columns, and dataset information.

📖 Introduction

Pandas is an open-source Python library used for data manipulation and analysis. It provides powerful data structures like Series and DataFrame, making it easy to work with structured data such as CSV files, Excel files, SQL databases, and more. Pandas is widely used in Data Science, Machine Learning, Artificial Intelligence, and Business Analytics.
